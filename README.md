### 說明
使用 useEffect, setInterval 做出簡單的計時器，可以自由調整工作時間、休息時間、組數，時間一到會播放短音效作為提醒。

### 技術
* React
* custom hook
* useReducer

### how to start

* `npm install`
* `npm start`
